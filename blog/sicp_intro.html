Plan for SICP

Posted: Fri, 23 Mar 2012
  • sicp

My plan to work on SICP and the resources that I will be relying on. I have been meaning to work through SICP for a long time and as I mentioned in my previous post, I am finally working on it. The plan right now is to work through it in clojure and if required later switch to scheme. I chose Clojure because the awesome Chris Stucchio told me it would be a good choice.

The resources that I plan to use are:

  • MIT Lecture Videos

  • The SICP Book(also available online.)

I will also keep posting the code I write on my github as well as keep posting solutions in the form of blog posts.

I will keep updating this section as I go along.
